# portal-hbzb-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式存放，便于通过 GitHub Pages 等方式直接访问。仓库不针对任何特定网站或域名，仅作为页面内容的整理与展示空间。

## 目录结构

```
portal-hbzb-notes/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放主要 HTML 页面文件
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 静态资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：包含所有独立归档的 HTML 页面。
- `assets/`：存放页面所需的样式、脚本、图片等资源。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可单独打开或部署。
- 页面之间不存在强制依赖关系，便于单独维护与更新。
- 归档内容可能包含示例、说明、工具页面等，不限于特定主题。

## 维护说明

- 欢迎提交 Pull Request 新增或更新页面。
- 请确保新增页面为自包含的 HTML 文件，并放在 `pages/` 目录下。
- 如需添加资源文件，请放入 `assets/` 对应子目录。
- 不建议在页面中引用外部不可控资源，以保证长期可用性。
